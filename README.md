geometeo (название просто от балды)

Умное погодное веб-приложение, которое не просто показывает температуру, но и помогает вам с помощью локального ИИ.

## Описание

Этот проект представляет собой веб-приложение для мониторинга погоды, использующее открытые данные **[Open-Meteo API](https://open-meteo.com/)**. Главная особенность проекта — интеграция локальной LLM (большой языковой модели). 

Вместо сухих цифр приложение анализирует текущие погодные условия и генерирует живые, персонализированные советы. Например: если на улице пасмурно и ожидается дождь, ИИ посоветует взять зонт и надеть ветровку.

##  Ключевые особенности

* **Точные погодные данные:** Интеграция с бесплатным Open-Meteo API.
* **Умные рекомендации:** Локальный ИИ анализирует погоду и дает советы по одежде и экипировке.
* **Конфиденциальность:** ИИ работает локально, ваши запросы не отправляются на сторонние серверы.
* **Адаптивный дизайн:** Удобный просмотр как на ПК, так и на мобильных устройствах.

##  Стек технологий

| Компонент | Технология |
| :--- | :--- |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Погодные данные** | [Open-Meteo API](https://open-meteo.com/) |
| **Локальный ИИ** | [Ollama](https://ollama.com/) / Llama.cpp + модель qwen3.8b |
